# Contributing to Jungle Grid

Thanks for your interest in contributing.

## How To Contribute

1. Fork the repository and create a feature branch.
2. Keep changes focused and small.
3. Add or update tests and documentation for behavior changes.
4. Open a pull request with a clear summary and motivation.

## Pull Request Guidelines

- Use descriptive commit messages.
- Link related issues in the pull request description.
- Include before/after details for user-visible changes.
- Ensure CI passes before requesting review.

## Reporting Issues

Please use the issue templates in `.github/ISSUE_TEMPLATE/` and include:

- Expected behavior
- Actual behavior
- Reproduction steps
- Relevant logs or screenshots

## Code Style

Follow existing style and conventions in the repository. Prefer readability and explicitness.

## Questions

If you are unsure about design or scope, open an issue first to discuss.
